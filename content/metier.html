Coeur de métier

Hébergeur de données
  • Sur la plateforme, la donnée devient FAIR
    • Facile à trouver
    • Accessible
    • Interopérable
    • Réutilisable
  • Maitrise des accès à chaque flux de données
    • Mode ouvert (opendata) ou fermé (privé)
    • Pour une réutilisation externe ou interne
Hébergeur de données
  • Diffusion de flux basés sur des fichiers tabulaires ou géographiques
  • Mise en place d'un contrat de service
  • Enrichissement possible
  • Portail ou catalogue de données
Diffuseur de données
  • Accéder aux données de référence en mode servicialisé
  • Mise à jour automatique, besoin limité de ressources pour la consommation
  • Base Sirène, Base d'adresses, données INSEE, cadastre, fonds de carte
  • Possibilité de rediffuser ces données filtrées
  • Contrat de service : stabilité
  • Normalisation de la description pour l'interopérabilité (OpenAPI 3.0)
Fournisseur d'applications
  • Applications métier liées aux données de référence et aux données hébergées
  • Légères pour fonctionner sur les terminaux mobiles
  • Founies en mode SaaS : mise a jour automatique, pas d'installation
  • Etape de configuration simple pour utiliser certains flux de données
    • Informations sur les parcelles
    • Fiches de données géolocalisées
    • Observatoire des entreprises
    • Portail de données
Principes de diffusion
  • Indexation textuelle (recherche fulltext) et géographique
  • Tuilage des données géographiques pour faciliter la consommation (adapté smartphone)
  • Sémantiser la donnée, vocabulaire réduit de concepts
  • L'interopérabilité met en valeur la donnée
  • La donnée doit arriver au plus vite
Prestations
  • Développements d'applications spécifiques
    • Besoins en interne
    • Communication
    • Réutilisation en marque blanche
  • Consulting, formation
  • Support des produits opensource
