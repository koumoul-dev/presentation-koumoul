Le métier

Diffusion de données
  • Accéder aux données de référence en tant que services
    • Base Sirene
    • Base d'Adresses Nationale
    • Données territoriales
    • Cadastre
    • Fonds de carte
  • Mise à jour automatique, besoin limité de ressources pour la consommation
Principes de diffusion
  • Stabilité
    • Contrat de service
    • Compatibilité ascendante
  • Interopérabilité
    • Normalisation de la description (OpenAPI 3.0)
    • Sémantiser la donnée, vocabulaire réduit de concepts
  • Rapidité
    • Recherche textuelle, par zone géographique, ...
    • Tuilage des données géographiques
    • Transmettre seulement les données nécessaires
Hébergement de données
  • Charger ses données sur la plateforme pour les diffuser
  • Flux basés sur des fichiers tabulaires ou géographiques
  • Enrichissement possible à partir d'autres flux
  • Portail ou catalogue des flux de données
  • Possibilité de rediffuser les données de référence filtrées
Hébergement de données
  • Sur la plateforme, la donnée devient FAIR
    • Facile à trouver
    • Accessible
    • Interopérable
    • Réutilisable
  • Maitrise des accès à chaque flux de données
    • Mode ouvert (opendata) ou fermé (privé)
    • Pour une réutilisation externe ou interne
Fourniture d'applications
  • Pour que la donnée arrive à l'usager
  • Applications métier liées aux données de référence et aux données hébergées
  • Légères pour fonctionner sur les terminaux mobiles
  • Founies en mode SaaS : mise a jour automatique, pas d'installation
    • Informations sur les parcelles
    • Fiches de données géolocalisées
    • Observatoire des entreprises
    • Portail thématique
